# v1.3.0 Release Notes

Release date: 2026-03-11

Release title: Visual Language Refresh + X Visual Lab

## Highlights

- Added `X Visual Lab` panel with six custom intelligence visuals:
  - Mentions velocity with spike markers
  - Quote impact funnel
  - Supporter retention trend
  - Concentration curve (actual vs equal-share baseline)
  - Supporter cohort heatmap
  - Post half-life distribution histogram
- Upgraded Day+Hour matrix cards with:
  - stronger contrast
  - top-window badges (`#1`, `#2`, `#3`)
- Upgraded overview chart style with:
  - insight overlays + sparkline strips
  - cleaner tooltip cards
  - lollipop ranking visuals for LinkedIn content charts
- Refined `crazy` theme colors and card interactions.

## Why this matters

- The dashboard now surfaces patterns faster without reading multiple tables.
- Visuals are less generic while preserving readability for teams.
- Better operator workflow for timing, supporter quality, and funnel health.

## Compatibility

- No migration required.
- Works with existing CSV/API setup from v1.2.
