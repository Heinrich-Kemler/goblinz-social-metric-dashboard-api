# v1.4.0 Release Notes

Release date: 2026-03-12

Release title: Insight Pack (Topics + Source Mix + Content-Type Time Windows)

## Highlights

- Mentions Intelligence now includes:
  - mention source mix (reply-style, link, hashtag, plain)
  - topic/hashtag leaderboard from mention text
- Added content-type best posting windows panel:
  - X (inferred content types from text/link pattern)
  - LinkedIn (exported content type labels)

## Why this matters

- You can see what kind of conversation is driving mentions, not just volume.
- You can map timing strategy to content format instead of using one generic best-time table.

## Notes

- X content type is heuristic in v1.4 (`Link`, `Hashtag-led`, `Hashtag`, `Short text`, `Text`).
- LinkedIn content-type windows depend on `linkedin_posts*.csv` with `Created date` and `Content Type`.
