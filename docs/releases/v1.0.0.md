# v1.0.0 Release Notes

Release date: 2026-03-11

## Highlights

- Hybrid ingestion model: CSV baseline + optional API enrichment per platform.
- Independent platform modes so X and LinkedIn can run in API/CSV combinations.
- Manual refresh flow with cost guardrails (cooldown, daily cap, in-flight lock).
- Advanced X intelligence panels:
  - Mentions intelligence
  - Quote analytics
  - Repeat supporters leaderboard (with verified filter)
  - Supporter retention (week-over-week)
  - Engagement cohort matrix
  - Engagement concentration (top-10/top-20 share + curve)
  - Follower snapshot tracking
  - Optional brand listening
- CSV validator and data-quality diagnostics.
- Non-technical onboarding assets: `IMPORT_GUIDE.md`, `PROMPT.md`, and metric references.

## Safety and Ops

- API credentials are server-side only (`.env.local`).
- `Data/cache/x_api_state.json` stores refresh/follower snapshot state only (no API keys).
- Added release process and rollback guide in `docs/RELEASE_PROCESS.md`.

## Defaults

- Mentions lookback defaults to one month: `X_MENTIONS_LOOKBACK_DAYS=30`.
- API usage defaults to manual refresh-first operation for cost control.

## Upgrade Guidance

- Stable users can pin: `git checkout v1.0.0`.
- Users who want newest fixes can track `main` and run `git pull origin main`.
